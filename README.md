# DomusMind

**DomusMind** is an open project to reduce the invisible mental load of managing a household.

Families coordinate a huge number of things every day: schedules, areas, tasks, routines, children, pets, events, logistics, and planning. Most of this coordination is informal, fragmented, and cognitively expensive.

DomusMind aims to make the **operating system of a household explicit**.

---

## The Idea

A home is a **system of time, coordination, and lightweight organization**.

DomusMind models that system with three core bounded contexts and one supporting concept:

- **Family** — household structure and members
- **Calendar** — events and shared time
- **Tasks** — execution of work and routines
- **Areas** — optional classification for household activity

The goal is simple:

> make household coordination visible, shared, and manageable.
