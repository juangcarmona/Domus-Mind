# DomusMind — System Overview

DomusMind helps a household coordinate family structure, plans, tasks, and routines through a shared operational view.

## Core Model

- Family — household identity and members
- Calendar — plans, dates, participants, reminders
- Tasks — actionable work and routines
- Areas — optional lightweight classification for activity

## Areas

Areas are flat, optional, and lightweight.
They improve scanning, grouping, colors, and filtering in timeline-oriented views.
They do not introduce workflow, ownership, or new domain rules.
