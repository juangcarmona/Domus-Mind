# DomusMind — Core Capabilities

## Family

- create a family
- add members
- view member details
- update family settings

## Calendar

- schedule plans/events
- reschedule plans/events
- cancel plans/events
- manage participants and reminders
- view family timeline

## Tasks

- create tasks
- assign and complete tasks
- manage routines
- view weekly coordination

## Areas

- classify entries with optional lightweight areas
- group and filter views by area where available
- use colors to improve recognition
